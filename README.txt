This dataset includes raw neuroimaging data from the neuroimaging of primary trigeminal neuralgia study, collected at Federal Neurosurgical Center, Novosibirsk, Russia. 
Data were collected between 2022 and 2024. Participants were adults with diagnosed primary trigeminal neuralgia. 
MRI data were collected before surgical treatment.  
This dataset includes precontrast 3D_TFE, 3D_T2w, DWI, and resting-state bold-fMRI.  
Anatomical scans were defaced using pydeface.
